MedicsWorld Website is a responsive, modern website designed to promote community health, mental wellness, maternal and child care, and public health initiatives. This project showcases my skills in **frontend web development** using **HTML, CSS, Bootstrap, and JavaScript**.


 Live Demo
[Visit MedicsWorld Live](https://admirable-tiramisu-dec472.netlify.app/)


 Technologies Used
- Frontend:** HTML5, CSS3, JavaScript (ES6+), Bootstrap 5  
- Design & UI: Responsive design, Figma-inspired layout  
- Libraries & Plugins: 
  - AOS (Animate on Scroll)  
  - Swiper.js (slider functionality)  
  - GLightbox (lightbox for media)  
  - Bootstrap Icons & Font Awesome  
- Version Control: Git & GitHub  


Features
- Fully responsive layout compatible with desktop, tablet, and mobile.  
- Hero section with impactful messaging and visual imagery.  
- "Why Us" section with interactive slider showcasing the organization's mission, vision, and values.  
- Services section highlighting key public health initiatives.  
- Call-to-Action section emphasizing community involvement and partnerships.  
- Blog section for news and updates.  
- Footer with social links and contact information.  

 Design Focus
- Clean, modern UI emphasizing readability and accessibility.  
- Smooth animations and transitions to engage users.  
- Mobile-first approach to ensure a seamless experience on all devices.
